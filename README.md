# Dental Clinic Management Platform 🦷

A full-stack web application designed to streamline dental clinic operations, enabling patients to book appointments, make payments, and interact with an AI voice assistant for support.

---

## 🚀 Features

- 🔐 Secure authentication system (Clerk)
- 📅 Appointment scheduling and management
- 💳 Payment integration for seamless transactions
- 🤖 AI voice assistant for real-time user support
- 📊 Modern, responsive UI for patients and clinic staff

---

## 🧠 Tech Stack

**Frontend / Full-stack**
- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui

**Backend / Infrastructure**
- Prisma ORM
- PostgreSQL
- Clerk Authentication

---

## 🏗️ Architecture Overview

- Modular Next.js app structure
- Role-based authentication flow via Clerk
- Prisma-powered database layer for type-safe queries
- Integrated payment workflow
- AI voice assistant embedded into user interface for support interactions

---

## 🎯 Purpose

This project demonstrates real-world full-stack development skills including authentication flows, payment systems, database design, and AI integration in a production-style web application.
